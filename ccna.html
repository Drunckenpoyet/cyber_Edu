Course Overview

Introduction to networking, LAN switching, routing protocols (OSPF, RIP), IP addressing, subnetting, network security (ACLs, VPNs), WAN technologies , and automation concepts.

Curriculum

  • Network Fundamentals: Basic networking concepts, OSI model, TCP/IP, and IP addressing.
  • LAN Switching: VLANs, trunking, STP, and network design.
  • Routing Technologies: Static and dynamic routing, OSPF, RIP, and EIGRP.
  • IP Addressing and Subnetting: IPv4, IPv6, subnet masks, and address planning.
  • Network Security: Access control lists (ACLs), VPNs, and secure network designs.
  • WAN Technologies: PPP, Frame Relay, and broadband connections.
  • Automation and Programmability: Introduction to network automation and SDN.
