Course Overview

Linux is used as an embedded OS for a variety of applications, including household appliances, automotive entertainment systems and network file system appliances. Network OS for routers, switches, domain name system servers, home networking devices and more.

Curriculum

  • Servers: Linux is the primary OS for servers.
  • Supercomputers: Linux runs on nearly all of the world's 500 fastest supercomputers.
  • Embedded systems: Linux is used in household appliances, automotive entertainment systems, and network file system appliances.
  • Network OS: Linux is used for routers, switches, domain name system servers, and home networking devices.
  • Smartphones: Linux is the basis for Android, which is used in smartphones.
